# Carousel

This is a carousel component that extends the functionality of the `Carousel` component from the `shadcn/ui` library.

```jsx copy
import {
  Carousel,
  CarouselMainContainer,
  SliderMainItem,
} from "@/components/extension/carousel";

export const CarouselExample = () => {
  return (
    <Carousel>
      <CarouselMainContainer>
        {Array.from({ length: 10 }).map((_, index) => (
          <SliderMainItem
            key={index}
            className="border border-muted flex items-center justify-center h-52 rounded-md"
          >
            Slide {index + 1}
          </SliderMainItem>
        ))}
      </CarouselMainContainer>
    </Carousel>
  );
};
```

## API Reference

### Props

The Carousel component accepts the following props:

| Prop            |            Type            | Default value |
| :-------------- | :------------------------: | ------------: |
| carouselOptions |    EmblaCarouselOption     |            -- |
| activeKeyboard  |          boolean           |         false |
| orientation     | "horizontal" or "vertical" |  "horizontal" |

### EmblaCarouselOption

For more information about the `EmblaCarouselOption` interface, please refer to the [Embla Carousel documentation]().

### activeKeyboard

This prop is used to enable or disable the keyboard navigation. When set to `true`, the keyboard navigation is enabled , else it is disabled.

### orientation

This prop is used to set the orientation of the carousel. It accepts two values: "horizontal" and "vertical". The default value is "horizontal".

## Accessibility

The Carousel component is accessible by default. It supports keyboard navigation and screen readers.

| key          |            Description             |
| :----------- | :--------------------------------: |
| `ArrowRight` | Increments by one step horzintanly |
| `ArrowLeft`  | Decrements by one step horzintanly |

## Usage Examples

### Carousel with navigation buttons

    ```jsx
    import { Carousel , CarouselMainContainer , SliderMainItem , CarouselPrevious , CarouselNext} from "@/components/extension/carousel";

    export const CarouselExample = () => {
      return (
        <Carousel activeKeyboard className="max-w-xs w-full ">
          <CarouselNext />
          <CarouselPrevious />
          <CarouselMainContainer>
            {Array.from({ length: 10 }).map((_, index) => (
              <SliderMainItem
                key={index}
                className="border border-muted flex items-center justify-center h-52 rounded-md"
              >
                Slide {index + 1}
              </SliderMainItem>
            ))}
          </CarouselMainContainer>
        </Carousel>
      );
    };
    ```
