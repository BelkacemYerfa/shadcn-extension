Welcome to Shadcn-extension-component documentation! This is a guide to help you get started with Shadcn-extension-component. Shadcn-extension-component is an extend for Shadcn/ui component library that provides a set of components for building web applications. It is built on top of [Shadcn/ui](https://ui.shadcn.com/).

## Motivation

Shadcn-extension-component is a set of components that are built on top of Shadcn-ui . It extends the component that are provided by shadcn and use the one there to build some common component for the most web apps . It is built with the following goals in mind:

- **Customizable**: Shadcn-extension-component is designed to be highly customizable. It provides a set of components that can be easily customized to fit your needs.
- **Easy to use**: Shadcn-extension-component is designed to be easy to use. It provides a set of components that are easy to use and can be easily integrated into your web application.
- **No need for installation**: Shadcn-extension-component does not require any installation. just copy past the code and install the shadcn/ui component that it uses.

## Getting Started

To get started with Shadcn-extension-component, you need to have a basic understanding of ReactJs/NextJs and Tailwinds . You also need to have a basic understanding of Shadcn-ui. If you are new to Shadcn-ui, you can check out the [Shadcn-ui documentation](https://ui.shadcn.com/).

## Installation

To install Shadcn-extension-component, you need to configure the shadcn/ui component that it uses. You can do this by following the instructions in the [Shadcn-ui documentation](https://ui.shadcn.com/).

## Usage

To use Shadcn-extension-component, you need to copy past the components that you want to use from the `shadcn-extension-component` . You can then use these components in your web application.

Example:

```jsx
import React from "react";

import { BreadCrumb } from "@/component/shadcn-extension-component";

export const BreadCrumbTest = () => {
  return (
    <BreadCrumb variant="ghost">
      <BreadCrumbItem>Home</BreadCrumbItem>
      <BreadCrumbSeparator />
      <BreadCrumbItem>Settings</BreadCrumbItem>
      <BreadCrumbSeparator />
      <BreadCrumbItem>Info</BreadCrumbItem>
    </BreadCrumb>
  );
};
```
