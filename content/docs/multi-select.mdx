A mulit select that assembles the missing features of the select element from [Shadcn ui](https://ui.shadcn.com/docs/components/select)

```jsx
import {
  MultiSelector,
  MultiSelectorContent,
  MultiSelectorInput,
  MultiSelectorItem,
  MultiSelectorList,
  MultiSelectorTrigger,
} from "@/components/extension/fancy-multi-select/multi-select-api";

export const MultiSelectTest = () => {
  const options = [
   /*array values*/
  ];
  const [value, setValue] = useState<string[]>([]);
  return (
      <MultiSelector
        value={value}
        onValueChange={setValue}
      >
        <MultiSelectorTrigger>
          <MultiSelectorInput placeholder="Select your framework" />
        </MultiSelectorTrigger>
        <MultiSelectorContent>
          <MultiSelectorList>
            {options.map((option, i) => (
              <MultiSelectorItem key={i} value={option.value}>
                {option.label}
              </MultiSelectorItem>
            ))}
          </MultiSelectorList>
        </MultiSelectorContent>
      </MultiSelector>
  );
};

```

## API Reference

### Props

The MultiSelect component accepts the following props:

| Prop     |          Type          | Default value |
| :------- | :--------------------: | ------------: |
| value    |        string[]        |            -- |
| setValue | (value : string)=>void |            -- |
| loop     |        boolean         |         false |

### Value

The state of the selected items

### setValue

A function that will be called when the value of the MultiSelect changes

### loop

A boolean that will determine if the list of selected items will loop or not when the user navigates with the arrow keys

#### Examples

```jsx {21} copy
import {
  MultiSelector,
  MultiSelectorContent,
  MultiSelectorInput,
  MultiSelectorItem,
  MultiSelectorList,
  MultiSelectorTrigger,

export const MultiSelectTest = ({ options } : MultiSelectProps) => {

        </MultiSelectorTrigger>
        <MultiSelectorContent>
          <MultiSelectorList>
            {options.map((option, i) => (
              <MultiSelectorItem key={i} value={option.value}>
                {option.label}
              </MultiSelectorItem>
            ))}
          </MultiSelectorList>
        </MultiSelectorContent>
      </MultiSelector>
  );
};
```

## Accessibility

The MultiSelect component is built with accessibility in mind. It is fully keyboard navigable and screen reader friendly.

### Keyboard Navigation

| key                     |             Description             |
| :---------------------- | :---------------------------------: |
| `Escape`                |           closes the list           |
| `Enter`                 |           opens the list            |
| `Space`                 |           opens the list            |
| `ArrowRight`            | Increments by one step horzintanly  |
| `ArrowLeft`             | Decrements by one step horzintanly  |
| `Delete` or `Backspace` | Removes the selected item from list |
